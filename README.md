# Academy AppSec  
Plataforma open source para estudar Segurança de Aplicações através de trilhas guiadas, conteúdo prático e colaboração da comunidade.

---

## Sobre o Projeto

A **Academy AppSec** nasceu com o objetivo de facilitar a jornada de quem deseja aprender **Application Security (AppSec)** de forma estruturada.

A plataforma organiza conteúdos, ferramentas, desafios e materiais open source em **trilhas de aprendizado**, cobrindo temas como:

- SAST, DAST, SCA, Iac;
- DevSecOps e automações;  
- Secure Coding & Modelagem de Ameaças.

Com foco em **aprendizado prático**, conteúdo gratuito e colaboração da comunidade.
